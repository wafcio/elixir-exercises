# Elixir Exercises

## Overview

This repo contains programming exercises in Elixir language.

- [Project Euler](https://projecteuler.net/)
- [Advent of Code](https://adventofcode.com/)

## Check code style

```
mix credo --all
```

## Verification

Every exercise contains test written in ExUnit.

```
mix test
```

### Check test coverage

```
mix test --cover
```
